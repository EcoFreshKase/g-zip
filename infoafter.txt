Thank you for using g-zip.

Right-click a file you want to convert to a DNA-sequence. Click "start g-zip". Configure the conversion to your liking and click convert. Congratulation, you have now converted you first file to a DNA-sequence.
To convert a DNA-sequence to a binary file. Follow the same steps.

Check the github repository of g-zip for further information on how to use g-zip: https://github.com/EcoFreshKase/g-zip
